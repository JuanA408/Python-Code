Description: 
* This program compares two PDF files by comparing bits.
* The files will be read in chunks as bits and stored as a has.
* When the bits are finalized being compared, a message will be output 
* to indicate if the files are identical. This program is run on VSCode terminal.

How to Run:
* Make sure the program is execute in the directory containing the CompareFiles.py file.
* There are some sample PDF files included.
